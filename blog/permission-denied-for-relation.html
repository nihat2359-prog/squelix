Permission Denied for Relation in PostgreSQL

What Causes Permission Denied?

This error appears when a user lacks sufficient privileges to access a table or view.

Grant Permissions

GRANT SELECT ON table_name TO username;

Check Current Privileges

\dp table_name

Schema Permissions

Users must also have USAGE permission on the schema.
